# Checklist Android V2

Esta versão abre o mesmo endereço do checklist em uma aba segura do Chrome.

Assim, no mesmo aparelho e no mesmo perfil do Chrome:
- o link e o aplicativo usam os mesmos dados;
- as alterações aparecem nos dois;
- o relatório funciona como no navegador;
- o nome exibido no aplicativo é apenas "Checklist".
