# Checklist Android V5

Versão Android com o checklist funcionando dentro do aplicativo.

Recursos:
- WebView com a página completa do checklist;
- botão nativo "Abrir versão web";
- visualização do relatório dentro do aplicativo;
- importação de backup por seletor de arquivos;
- navegação pelo botão voltar;
- nome genérico "Checklist".

Página carregada:
https://wagnerliborioporto.github.io/checklist-wagner/checklist.html

Observação:
O armazenamento do WebView é separado do armazenamento do Chrome.
Para sincronização automática entre aplicativo, navegador e outros dispositivos,
é necessário usar banco de dados em nuvem.
